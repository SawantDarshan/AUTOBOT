# Contributing to AUTOBOT

Thanks for your interest in contributing! Here's how to get involved.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-improvement`)
3. Make your changes
4. Commit with a clear message (`git commit -m "Add: description of change"`)
5. Push to your fork and open a Pull Request

## Guidelines

- Keep changes focused and minimal
- Follow existing code style and conventions
- Update documentation if your change affects usage
- Be respectful and constructive in discussions

## Reporting Issues

Open a GitHub issue with:
- Clear description of the problem
- Steps to reproduce (if applicable)
- Expected vs actual behavior

## Ideas for Contributions

- New diagnostic categories
- Language/framework-specific detection rules
- Improved reporting formats
- Safety guardrail improvements
